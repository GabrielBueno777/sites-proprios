═════════════════════════════════════════════════════════════════════════════════
                        COMO COMEÇAR (3 PASSOS)
═════════════════════════════════════════════════════════════════════════════════

PASSO 1: INSTALAR
─────────────────
Windows:
  Double-click: setup.bat

Linux/macOS:
  chmod +x setup.sh && ./setup.sh

Manual:
  npm install

PASSO 2: RODAR
──────────────
npm run dev:watch

(Melhor para desenvolvimento com hot-reload)

PASSO 3: ACESSAR & COMPARTILHAR
─────────────────────────────────
Local:
  http://localhost:3000

Rede (Compartilhe este):
  http://<SEU_IP>:3000
  Ex: http://192.168.1.100:3000

Internet (Com ngrok):
  ngrok http 3000
  https://xxxx-xx-xxx.ngrok.io

   • Configure perfil
   • Veja treino/dieta

6. Descobra seu IP:

   Linux:   ifconfig

7. Compartilhe:
   http://<SEU_IP>:3000

8. Outros acessam o link!

═════════════════════════════════════════════════════════════════════════════════
                        TROUBLESHOOTING RÁPIDO
═════════════════════════════════════════════════════════════════════════════════

❓ Porta 3000 em uso?
   Mude para: PORT=8080 npm run dev:watch

❓ Node não instalado?
   Instale em: https://nodejs.org

❓ npm install falha?
   npm cache clean --force && npm install

❓ Não consegue ver servidor?
   curl http://localhost:3000

❓ CORS não funciona?
   Verifique se server.ts está rodando
   Reinicie servidor

🌐 COMO COMPARTILHAR

1️⃣ REDE LOCAL (Mesmo WiFi/Ethernet)
   • Descobra seu IP: ipconfig (Windows) ou ifconfig (Linux)
   • Compartilhe: http://SEU_IP:3000
   • Exemplo: http://192.168.1.100:3000

2️⃣ INTERNET (Ngrok - RECOMENDADO)
   • Instale: ngrok.com
   • Execute: ngrok http 3000
   • Compartilhe o link gerado
   • Mais seguro que port forwarding

3️⃣ DEPLOY PERMANENTE
   • Heroku: heroku create && git push heroku main
   • Vercel: Conecte repositório
   • AWS: EC2 + Elastic IP

🆘 TROUBLESHOOTING

Porta em uso?
  Windows: netstat -ano | findstr :3000 && taskkill /PID <PID> /F
  Linux:   lsof -i :3000 && kill -9 <PID>

Node não encontrado?
  Instale em: https://nodejs.org

npm install falha?
  npm cache clean --force && npm install

CORS ainda com problema?
  Verifique se server.ts está ativo
  curl -i http://localhost:3000

🆘 SOLUÇÃO RÁPIDA

Se não funcionar:

1. Porta em uso?
   Windows: netstat -ano | findstr :3000
   Linux:   lsof -i :3000

2. Node não está instalado?
   Baixe em: https://nodejs.org

3. npm install falha?
   npm cache clean --force && npm install

4. Não consegue ver o servidor?
   curl http://localhost:3000

5. Quer usar porta diferente?
   PORT=8080 npm run dev:watch

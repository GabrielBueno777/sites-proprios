❓ PERGUNTAS FREQUENTES

P: Preciso parar o servidor antigo?
R: Não, eles podem coexistir em portas diferentes

P: Onde está meu código TypeScript?
R: Em server.ts (o JavaScript antigo ainda está em server.js)

P: Como compartilho com pessoas fora da rede?
R: Use ngrok (seguro) ou port forwarding (arriscado)

P: Preciso compilar antes de rodar?
R: Não, use npm run dev (ts-node compila na memória)

P: Em produção, preciso compilar?
R: Sim, use npm run build && npm start
